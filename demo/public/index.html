CSS color conversion reference

Convert RGB, HEX, RGBA, HSL, and HSV colors

Color Tools is built for quick CSS color work: paste a value, copy the equivalent formats, then keep moving into contrast checks, gradients, harmonies, tints, shades, tones, and palettes. The converter supports common designer and developer workflows including RGB to HEX, HEX to RGB, HSL to HEX, HSV to RGB, and RGBA values with alpha.

Every demo capability is backed by the color-tools Clojure library, including conversion, contrast, palettes, gradients, harmonies, blending, Delta E, and Kelvin color temperature helpers. Read the cljdoc documentation or browse the jramosg/color-tools repository.

What the demo includes

  • HEX, RGB, RGBA, HSL, and HSV conversion
  • WCAG AA and AAA foreground/background contrast checks
  • Palette generation for analogous, triadic, tetradic, and random schemes
  • Gradient interpolation in RGB, HSL, and HSV color spaces
  • Alpha compositing, blend modes, Delta E, and Kelvin to RGB

FAQ

Color converter questions

How do I convert RGB to HEX?

Enter an RGB value such as rgb(52, 152, 219) in the color converter. The tool returns the equivalent HEX value, plus RGBA, HSL, and HSV formats.

How do I convert HEX to RGB?

Paste a HEX color such as #3498db into the converter. Color Tools returns rgb(52, 152, 219), plus matching RGBA, HSL, and HSV values.

Can I convert HEX to HSL?

Yes. Paste a HEX color such as #3498db and the converter shows the matching HSL, RGB, RGBA, and HSV values.

Is this color converter free?

Yes. Color Tools is free to use in the browser and is powered by an open-source Clojure and ClojureScript library.

Does the tool support alpha values?

Yes. The converter accepts rgba() values with alpha from 0 to 1 and keeps the alpha channel in the RGBA output.

What color tools are included?

Color Tools includes RGB, HEX, RGBA, HSL, and HSV conversion, WCAG contrast checking, gradient generation, palette generation, color harmonies, blending modes, tints, shades, tones, Delta E, and Kelvin color temperature conversion.
